# Campus Notifications Platform

A notification management application built using React and Material UI.

## Features

- Notification feed
- Priority inbox
- Category filtering
- Read/unread tracking
- Local storage persistence
- Logging middleware integration

## Project Structure

logging_middleware/
notification_app_be/
notification_app_fe/
notification_system_design.md

## Running Frontend

cd notification_app_fe
npm install
npm run dev

## Running Backend

Follow the backend setup inside notification_app_be.
